# **MobTyper Pipeline**

# Overview:
This pipeline processes plasmid mobility typing results generated by MOB-suite (mobtyper). It summarizes replicon types, relaxase families, MPF types, and generates visualizations.

# Repository Structure:
mobtyper_pipeline/


1. scripts/
   ```
      mobtyper_analysis.R
   
   ```
2. documentation/
   ```
     mobtyper_analysis documentation.pdf
   ```

# Scripts:
mobtyper_analysis.R - Performs the following
1. Reads merged mobtyper output (mobtyper_merged.csv)
2. Cleans column names
3. Generates:
  ```
       Replicon summary table
       Relaxase summary table
       MPF summary table
  ```

4. Produces barplots for each category
5. Saves all outputs for downstream AMR or plasmid mobility analysis

# Expected Outcomes:
```
replicon_summary.tsv
relaxase_summary.tsv
mpf_summary.tsv
replicon_plot.png
relaxase_plot.png
mpf_plot.png
Rplots.pdf
```
